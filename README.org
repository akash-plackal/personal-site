* Hello, this is my personal website.

** One-time install (no repo npm deps)

#+begin_src sh
npm i -g html-minifier-next
#+end_src

** Build

#+begin_src sh
make build
#+end_src

Build output is written to =dist/=.
Default minification mode is =max= (smallest output). Use =safe= mode if needed:

#+begin_src sh
HTML_MINIFIER_MODE=safe make build
#+end_src

** Budget check only

#+begin_src sh
make check
#+end_src

** Local preview

#+begin_src sh
make serve
#+end_src

Then open =http://localhost:8080=.

** Cloudflare Pages

- Build command: =./scripts/build.sh=
- Build output directory: =dist=
